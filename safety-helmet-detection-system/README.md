# Safety Helmet Detection System

**Domain:** Computer Vision  
**Part of:** UT Austin PG Program in AI & ML

## Objective

Develop an image classification model to automatically detect helmet compliance in hazardous work environments.

## Key Focus Areas

- Image preprocessing
- CNN-based classification
- Deployment readiness
- Transfer learning (if used)

## How to Run

1. Open `HelmNet_Project.ipynb` in Jupyter Notebook or Google Colab. GPU is recommended for training.
2. Ensure `Labels_proj.csv` (and any image paths referenced in the notebook) are available. Image data may be loaded from a URL or external path as per the notebook instructions.
3. Install dependencies (e.g. `tensorflow`, `keras`). The notebook may include `pip install` cells.
4. Run all cells sequentially.

## Files in This Folder

| File | Description |
