# Personal Loan Propensity Modeling

**Domain:** Banking / Marketing Analytics  
**Part of:** UT Austin PG Program in AI & ML

## Objective

Predict which liability customers are most likely to purchase personal loans and identify key attributes driving conversion, enabling targeted marketing campaigns.

## Key Focus Areas

- Customer segmentation
- Feature importance & explainability
- Conversion-focused classification

## How to Run

1. Open `Machine_learning_Loan_problem.ipynb` in Jupyter Notebook or Google Colab.
2. Ensure `Loan_Modelling.csv` is in the same folder (included in repo).
3. Run all cells sequentially.

## Files in This Folder

| File | Description |
|------|-------------|
| `Machine_learning_Loan_problem.ipynb` | Main modeling notebook (EDA, preprocessing, classification, evaluation) |
| `Machine_learning_Loan_problem.html` | Exported HTML report |
| `Loan_Modelling.csv` | Banking/customer dataset |

## Key Libraries

- Python, Pandas, NumPy
- Scikit-learn (classification, preprocessing, metrics)
- Matplotlib, Seaborn
- Jupyter Notebook
